Heading Elements - There are only 6 heading tags from h1 to h6. h1 being the largest and h6 is the smallest.
The most important one is the H1 tag which represents a title or heading and it should be used once per page.

<h1> This is H1 heading Tag </h1>

<h2> This is H2 heading Tag </h2>

<h3> This is H3 heading Tag </h3>

<h4> This is H4 heading Tag </h4>

<h5> This is H5 heading Tag </h5>
<h6> This is H6 heading Tag </h6>
